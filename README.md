# Zootopia API

The Animals API provides interesting scientific facts on thousands of different animal species.
We use this API to make our own program to work with animals data.

## Installation

To install this project, simply clone the repository and install the dependencies in requirements.txt using `pip`

## Usage

To use this project, run the following command - `python3 animals_web_generator.py`.
